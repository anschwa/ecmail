# How to connect to Earlham's Zimbra Email Server

# Mail Settings
## IMAP:
- Server: zibra.ealham.edu
- Port: 143 or 993
- Use SSL / TLS

## SMTP:
- Server: ecmailer.earlham.edu
- Port: 465
- Use SSl / TLS
